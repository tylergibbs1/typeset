# Authentication

All API requests require a Bearer token.

```bash
Authorization: Bearer ts_test_full_<key>
```

## Key format

| Prefix | Environment | Scope |
|--------|-------------|-------|
| `ts_live_render_` | Production | Render only |
| `ts_live_extract_` | Production | Extract only |
| `ts_live_full_` | Production | Full access |
| `ts_test_full_` | Sandbox | Full access (watermarked) |

## Scopes

| Scope | Endpoints |
|-------|-----------|
| `render` | `/v1/render`, `/v1/render/async` |
| `extract` | `/v1/extract` |
| `template` | `/v1/templates`, `/v1/template-from-doc` |
| `full` | All endpoints |

## Rate limits

| Tier | Requests/min | Documents/month |
|------|-------------|-----------------|
| Free | 10 | 100 |
| Pro | 60 | 5,000 |
| Scale | 300 | Unlimited |

<Callout type="warn">
Keys are SHA-256 hashed before storage. Never log or expose raw API keys.
</Callout>
