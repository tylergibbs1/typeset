# POST /v1/render

Renders a document synchronously. Returns a signed URL to the generated PDF.

## Request

```json
{
  "template": "<h1>{{title}}</h1>",
  "data": { "title": "My Document" },
  "engine": "html",
  "format": "pdf",
  "options": {
    "pageSize": "a4",
    "orientation": "portrait",
    "margin": { "top": "20mm", "right": "15mm", "bottom": "20mm", "left": "15mm" },
    "smartLayout": true,
    "verify": false
  }
}
```

| Field | Type | Default | Description |
|-------|------|---------|-------------|
| `template` | string | required | HTML string, Typst string, or template ID (`tpl_xxx`) |
| `data` | object | `{}` | Data to inject into the template |
| `engine` | `"html"` \| `"typst"` | auto-detected | Rendering engine |
| `format` | `"pdf"` | `"pdf"` | Output format |
| `options.pageSize` | `"a4"` \| `"letter"` \| `"legal"` | `"a4"` | Page size |
| `options.orientation` | `"portrait"` \| `"landscape"` | `"portrait"` | Page orientation |
| `options.smartLayout` | boolean | `true` | AI layout optimization (HTML only) |
| `options.verify` | boolean | `false` | Run OCR verification after render |

## Response

```json
{
  "data": {
    "runId": "run_abc123",
    "url": "https://...",
    "expiresAt": "2026-04-02T03:00:00Z",
    "verification": {
      "status": "skipped",
      "score": 0,
      "issues": []
    },
    "metadata": {
      "pages": 1,
      "sizeBytes": 74659,
      "renderTimeMs": 516,
      "templateHash": "cef7ad...",
      "dataHash": "5d025d..."
    }
  }
}
```

## Headers

| Header | Description |
|--------|-------------|
| `Idempotency-Key` | Optional. Same key returns same response without re-rendering. |
| `Typeset-Version` | API version (default: `2026-04-01`) |

## Async variant

Use `POST /v1/render/async` for long-running renders. Returns immediately with a run ID. Supports webhooks.
