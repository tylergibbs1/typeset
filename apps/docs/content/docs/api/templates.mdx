# Templates

Templates can be saved and versioned for reuse across renders.

## POST /v1/templates

```json
{
  "name": "Monthly Invoice",
  "content": "<html>...</html>",
  "description": "Standard invoice template"
}
```

**Response (201):**

```json
{
  "data": {
    "id": "tpl_abc123",
    "version": 1,
    "hash": "cef7ad...",
    "createdAt": "2026-04-01T00:00:00Z"
  }
}
```

## GET /v1/templates

Lists templates with cursor pagination.

## GET /v1/templates/:id

Returns the full template including content.

## Using a template ID in render

```json
{
  "template": "tpl_abc123",
  "data": { "invoiceNumber": "INV-001" }
}
```
