# Getting Started

<Steps>
<Step>
### Install dependencies

```bash
git clone https://github.com/tylergibbs1/typeset.git
cd typeset
bun install
```
</Step>

<Step>
### Start infrastructure

<Tabs items={['Docker (recommended)', 'Supabase + Upstash']}>
<Tab value="Docker (recommended)">
```bash
docker compose up -d
```
This starts Postgres and Redis locally.
</Tab>
<Tab value="Supabase + Upstash">
Create a Supabase project and an Upstash Redis instance, then set the connection strings in `.env`.
</Tab>
</Tabs>
</Step>

<Step>
### Configure environment

```bash
cp apps/api/.env.example apps/api/.env
```

Fill in your keys:

| Variable | Required | Description |
|----------|----------|-------------|
| `DATABASE_URL` | Yes | Postgres connection string |
| `UPSTASH_REDIS_REST_URL` | Yes | Upstash Redis REST URL |
| `UPSTASH_REDIS_REST_TOKEN` | Yes | Upstash Redis token |
| `R2_ACCOUNT_ID` | Yes | Cloudflare account ID |
| `R2_ACCESS_KEY_ID` | Yes | R2 API key ID |
| `R2_SECRET_ACCESS_KEY` | Yes | R2 API secret |
| `OPENAI_API_KEY` | Optional | For AI layout engine |
| `MISTRAL_API_KEY` | Optional | For OCR verification + extraction |
</Step>

<Step>
### Push the database schema

```bash
cd packages/db
bunx drizzle-kit push
```
</Step>

<Step>
### Seed a test API key

```bash
bun scripts/seed.ts
```

This creates a test organization and API key: `ts_test_full_localdev0000000000000000`
</Step>

<Step>
### Start the API

```bash
cd apps/api
bun run dev
```

The API is now running at `http://localhost:3000`.
</Step>

<Step>
### Render your first document

```bash
curl -X POST http://localhost:3000/v1/render \
  -H "Authorization: Bearer ts_test_full_localdev0000000000000000" \
  -H "Content-Type: application/json" \
  -d '{
    "template": "<h1>Hello {{name}}</h1>",
    "data": {"name": "World"}
  }'
```
</Step>
</Steps>
